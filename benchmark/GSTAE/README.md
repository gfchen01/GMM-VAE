# Gate Stacked Target-Related Auto Encoder
## T-SAE
在与训练的过程中,加入标签数据,将无监督预训练转化为有监督预训练问题。其中引入的超参数λ需要自行调整
## GS-TAE
利用门控逻辑单元对中间的抽象特征进行集成进而得到y的数据

## 测试集性能评估
 软测量模型  | RMSE  | R2
 ----- | ----- | ------  
 T-SAE  | 0.0357 | 0.9518
 GS-TAE  | 0.0299 | 0.9704
